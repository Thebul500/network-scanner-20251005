# Network Scanner

A simple network port scanner for discovering open ports on target hosts.

## Features
- Fast port scanning
- Customizable port ranges
- Threading support
- Command-line interface

## Usage
```bash
python3 main.py <host> -s <start_port> -e <end_port>
```

## Example
```bash
python3 main.py 192.168.1.1 -s 1 -e 1000
```

## Security Note
This tool is for educational and authorized testing purposes only.
